# 計画: 日本株カレンダーと欠損日の扱いを仕様固定

## ゴール

- 日本株向けの価格前処理ポリシー（営業日・欠損値）を明文化し、コードとテストで再現可能に固定する。

## スコープ

- 対象:
  - 価格データ向けカレンダー/欠損ポリシーの実装
  - 仕様ドキュメント追加
  - 回帰テスト追加
- 対象外:
  - 祝日カレンダーの厳密実装（取引所休場日データ連携）
  - コーポレートアクション補正

## チェック

- Lint: `uv run ruff check .`
- 型検査: `uv run mypy`
- Tests: `uv run pytest -q`
- Windows 権限制約環境: `powershell -ExecutionPolicy Bypass -File scripts/run_quality_gate.ps1`

## 手順

1. 日本株向け価格ポリシーを `src` に実装する。
2. Raw CSV 変換スクリプトへポリシー適用を組み込む。
3. 仕様ドキュメントとテストを追加する。
4. 品質ゲートを実行し、Execute と状態ファイルを更新する。

## リスク

- リスク:
  - 週末除外のみでは祝日を判定できない
- 緩和策:
  - 仕様で「現時点は平日ベース」と明記し、将来拡張ポイントを残す
